# MED-DNA (MedDNA Premium Medical Companion)

This template provides a setup to run the MedDNA Premium client web application with React, TypeScript, and Vite.

## Local Development Setup

1. Install project dependencies:
   ```powershell
   npm install
   ```

2. Start the hot-reloading development server:
   ```powershell
   npm run dev
   ```
   Open `http://localhost:5173/` in your browser.

3. Compile the production bundle:
   ```powershell
   npm run build
   ```
